Wikimedia Toolforge

Wikimedia Toolforge is a free cloud hosting platform for Wikimedia contributors. It provides web servers, data access, job management, and other features to help developers maintain tools and bots.

Toolforge admin console helps Toolforge users manage their tools.

Toolforge is a part of the wider Wikimedia Cloud Services suite of services, supported by Wikimedia Foundation staff and movement volunteers.
